# DouyinFetch
## **抖音用户主页信息监控**
| 该工具可获取抖音用户主页信息, 如**头像**, **昵称**, **粉丝数**, **ip属地**, **个性签名**等信息

## 运行方式
### 环境要求
* Python>=3.12
### 本地部署

1. 克隆项目到本地
```
git clone https://github.com/1716285375/DouyinFetch.git
```
2. 安装依赖包
```
pip install -r requirements.txt
```
3. 运行 main.py
```
python main.py
```
4. 打开 index.html
```
4.1 输入要查询的用户的抖音号
4.1.1 复制用户的抖音号
4.1.2 在输入框中粘贴该抖音号
4.2 点击查询
```
